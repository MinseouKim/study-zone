01 : 인터넷과 웹(Internet& Web)

1: 인터넷의 탄생

인터넷 = 전세계를 연결하는 고속도로

웹 = 인터넷이라는 고속도로를 따라 연결된 도시에 생겨난 건물

1.통신의 시작

통신이란?
의사, 감정, 정보를 한쪽에서 다른 쪽으로 전달하는 것

데이터 통신과 프로토콜
데이터 통신 : 컴퓨터와 같은 기기들끼리 데이터를 주고받는 것
프로토콜 : 통신 약속을 정해주는 것 ex)누가 먼저 데이터를 보낼것인지

컴퓨터 네트워크에서 데이터를 전송하기 위해서 필요한 조건 2가지
1.물리적인 연결(유무선 통신)
2.통신 약속(프로토콜)
